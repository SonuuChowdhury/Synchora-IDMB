# Synchora Server Node.js Project

This is a basic Node.js project setup for Synchora Server.

## How to Run

1. Make sure you have Node.js installed.
2. In the project directory, run:
   
   ```powershell
   node index.js
   ```

You should see:

```
Hello, Synchora Server!
```
